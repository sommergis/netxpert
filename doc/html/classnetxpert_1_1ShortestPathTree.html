Detailed Description

Solver for the Shortest Path Tree Problem.

  • Initialization with a netxpert::cnfg::Config object in Constructor
  • call of SetOrigin() method
  • optional: call of SetDestinations() method defines 1-n destinations (if no destination is set with this method, the whole shortest path tree is computed from the origin)
  • call of Solve( netxpert::data::InternalNet ) method for the computation; the network must be constructed prior to this step
  • optional: GetOptimum() for getting the overall optimum
  • optional: GetShortestPaths() for getting the actual result of the SPT computation
  • optional: SaveResults() or alternate GetResultsAsJSON()

all other setters and getters can be used for overriding the configuration from the constructor
