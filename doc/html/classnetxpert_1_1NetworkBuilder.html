Detailed Description

Builds a network of the given linestrings.

  • Multipart to Single Part Linestrings

StartPoint(Linestring): FromNode, EndPoint(Linestring): ToNode

  • Undirected als Standard in Solver input –> Pro (compared to planarization): no loss of attribute information on the edges of the graph –> Con (compared to planarization): much more (useless) egdes and nodes in the resulting graph –> IN: lines –> OUT: edges in network structure with FromNode and ToNode and all input attributes
