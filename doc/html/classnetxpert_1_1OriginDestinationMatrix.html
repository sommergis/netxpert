Detailed Description

Solver for computing an Origin Destination Matrix.

Todo:
Check for duplicate code -> SPT Solver - Inheritance?
  • Initialization with a netxpert::cnfg::Config object in Constructor
  • call of SetOrigins() method defines 1-n origins (must be at least 1 origin)
  • call of SetDestinations() method defines 1-n destinations (must be at least 1 origin)
  • call of Solve( netxpert::data::InternalNet ) method for the computation; the network must be constructed prior to this step
  • optional: GetOptimum() for getting the overall optimum
  • optional: GetODMatrix() for getting the actual result of the ODM computation
  • optional: SaveResults()

all other setters and getters can be used for overriding the configuration from the constructor
