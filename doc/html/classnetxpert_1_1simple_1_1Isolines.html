Detailed Description

Simple Interface of the Isolines Solver.

Warning
EXPERIMENTAL!
  • Initialization with a JSON Config in Constructor
  • call of Solve() method (saves the results to the ResultDB given in the config JSON
  • call SetCutOffs() to set the distance cut off values
  • optional: GetOptimum() returns the overall optimum
  • optional: GetShortestPathsAsJSON() returns the shortest paths as JSON string
