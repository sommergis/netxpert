Detailed Description

Simple Interface of the MinCostFlow Solver.

  • Initialization with a JSON Config in Constructor
  • call of Solve() method (saves the results to the ResultDB given in the config JSON)
  • optional: GetOptimum() returns the overall optimum
  • optional: GetMinimumCostFlowAsJSON() returns the minimum cost flow as JSON string
