For saving a subset of original arcs and addintional route parts in the original netXpert DB. This method performs a SELECT of the arcs (arcIDs) in the original netXpert DB and inserts them in a new table (data processing within the DB). Obviously this is faster and memory friendlier than reading the arcs into memory first and push them into a database afterwards (geometry parsing etc.).
