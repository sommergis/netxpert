void netxpert::Transportation::Solve ( )

Computes the transportation problem.

Solves the Transportation Problem with the predefined ODMatrix and NodeSupply property. Solves on pure attribute data only i.e. output is always without geometry. (GEOMETRY_HANDLING is always set to "NoGeometry")

void netxpert::Transportation::Solve ( netxpert::data::InternalNet &  net)
virtual

Computes the transportation problem on the given network.

Solves the Transportation Problem with the given network and all origin and destination nodes. Uses the netXpert OriginDestinationMatrix Solver internally.

Reimplemented from netxpert::MinCostFlow.
