Detailed Description

Solver for the Minimum Cost Flow Problem.

  • Initialization with a netxpert::data::InternalNet object in Constructor
  • call of Solve( netxpert::data::InternalNet ) method for the computation; the network must be constructed prior to this step
  • optional: GetSolverStatus() for getting the status of the solver
    See also
    netxpert::data::MCFSolverStatus
  • optional: GetOptimum() for getting the overall optimum
  • optional: GetMinCostFlow() for getting the actual result of the MCF computation
  • optional: SaveResults()

all other setters and getters can be used for overriding the configuration from the constructor
